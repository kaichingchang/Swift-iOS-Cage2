# Swift-iOS-Cage2

這是從 macOS App 移植到 iOS 的困獸籠遊戲，模擬動物被關在籠子裡面，籠子有四個位置，使用者可用上、下、左、右方向鍵挪移動物的位置，並且幫動物找到出口脫逃。

> macOS App 專案網址 [https://github.com/kaichingchang/Swift-macOS-Cage](https://github.com/kaichingchang/Swift-macOS-Cage)
